# Install Pipenv

`pip install pipenv`

## Install Dependencies

`pipenv install`

## Activate virtualenv

`pipenv shell`

## Deactivate project

`ctrl+D or exit`

# API

- localhost:8000/
  - api/test -> test purpose
  - api/company ->GET(show all), POST(entry a company)
  - api/company/1 ->GET(company details), PATCH(update company), DELETE(delete a company)
